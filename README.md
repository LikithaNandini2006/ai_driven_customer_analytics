# AI-Driven Customer Analytics Platform
DEMO LINK:https://ai-driven-customer-analytics-3.onrender.com
## Overview

AI-Driven Customer Analytics Platform is a Machine Learning-based web application developed using **Python, Streamlit, Pandas, Scikit-Learn, and Data Visualization techniques**. The system helps businesses analyze customer behavior, identify customer segments, predict customer churn, forecast future purchases, and generate personalized product recommendations.

The platform provides an interactive dashboard that transforms customer data into actionable business insights.

## Machine Learning Models

### 1. Customer Segmentation

Algorithm Used:

* K-Means Clustering

Purpose:

* Group customers into different segments based on purchasing behavior.

### 2. Churn Prediction

Algorithm Used:

* Logistic Regression

Purpose:

* Predict whether a customer is likely to leave.

### 3. Purchase Prediction

Algorithm Used:

* Random Forest Classifier

Purpose:

* Predict future customer purchases.

### 4. Recommendation System

Purpose:

* Suggest products based on customer preferences.

## Installation

### Clone Repository

```bash
git clone https://github.com/LikithaNandini2006/ai_driven_customer_analytics.git
```

### Navigate to Project Folder

```bash
cd ai_driven_customer_analytics
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
